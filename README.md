> **Terra UST Autobuild** provides the automated build and deployment pipeline for the Universal Semantic Translator across the FMP ecosystem.

## 🔄 Pipeline

```
Source repos → Terra Audit → UST Build → Validation → Deploy
```

## 🛠️ Features

- **Automated Terra Audit** — validates all FMP repositories
- **UST compilation** — assembles semantic components
- **GitHub Pages deploy** — live deployment on commit
- **Cross-repo sync** — keeps FMP ecosystem consistent
